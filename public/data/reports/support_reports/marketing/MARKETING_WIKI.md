# Marketing Data Wiki

## 1. Overview
This document serves as the **Single Source of Truth** for the Marketing Data Layer. It defines the physical tables available in the system and how they map to the "Logical Data Keys" requested by the Business Support Analytics Engine.

## 2. Logical to Physical Mapping

| Logical Key (Report Requirement) | Physical Table ID | Description |
| :--- | :--- | :--- |
| **CAMPAIGNS**, `Promotions` | `MKT_Campaigns` | Marketing campaigns. |

## 3. Table Definitions

### 3.1 Campaigns

#### `MKT_Campaigns`
*Campaign initiatives.*
- **campaign_id** (`string`, PK)
- **campaign_name** (`string`)
- **status** (`string`): `Active`, `Planned`, `Completed`.
- **budget** (`number`)

### 3.2 Lead Gen

#### `MKT_Leads`
*Inbound leads.*
- **lead_id** (`string`, PK)
- **source_campaign_id** (`string`, FK)
- **status** (`string`): `New`, `Contacted`, `Qualified`.
