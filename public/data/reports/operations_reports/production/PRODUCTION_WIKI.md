# Production Data Wiki

## 1. Overview
This document serves as the **Single Source of Truth** for the Production Data Layer. It defines the physical tables available in the system and how they map to the "Logical Data Keys" requested by the Operations Analytics Engine.

## 2. Logical to Physical Mapping

| Logical Key (Report Requirement) | Physical Table ID | Description |
| :--- | :--- | :--- |
| **PRODUCTION_ORDERS**, `Manufacturing` | `PRD_Production_Orders` | Manufacturing orders. |
| **BOM**, `Recipe` | `PRD_Bill_Of_Materials` | Product components. |
| **WORK_CENTERS**, `Lines` | `PRD_Work_Centers` | Production lines. |
| **OUTPUT**, `Yield` | `PRD_Production_Log` | Daily production output. |
| **DOWNTIME**, `Stoppage` | `PRD_Downtime` | Line downtime events. |

## 3. Table Definitions

### 3.1 Planning

#### `PRD_Production_Orders`
*Manufacturing orders.*
- **order_id** (`string`, PK)
- **product_id** (`string`, FK)
- **quantity** (`number`)
- **due_date** (`date`)

#### `PRD_Bill_Of_Materials`
*Product recipes.*
- **bom_id** (`string`, PK)
- **product_id** (`string`, FK)
- **component_id** (`string`, FK)

### 3.2 Execution

#### `PRD_Production_Log`
*Production output.*
- **log_id** (`string`, PK)
- **qty_produced** (`number`)
- **qty_rejected** (`number`)

#### `PRD_Downtime`
*Line stoppages.*
- **event_id** (`string`, PK)
- **reason** (`string`): `Breakdown`, `No Material`, `Changeover`.
- **duration_minutes** (`number`)
